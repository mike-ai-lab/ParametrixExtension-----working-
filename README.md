# PARAMETRIX - SketchUp Layout Generator Extension

A powerful SketchUp extension for generating parametric layouts with advanced trimming and positioning capabilities.

## Features

- **Parametric Layout Generation**: Create complex tile and panel layouts with customizable parameters
- **Multi-Face Positioning**: Apply layouts across multiple faces simultaneously
- **Advanced Trimming**: Precise 3D trimming with enhanced accuracy
- **Preset Management**: Save and load layout presets for quick reuse
- **Mitered Rails**: Support for mitered rail implementations
- **License Management**: Built-in licensing system

## Installation

1. Download the extension files
2. Copy the entire folder to your SketchUp Plugins directory
3. Restart SketchUp
4. The PARAMETRIX toolbar should appear

## Directory Structure

- `lib/PARAMETRIX/` - Core extension modules
- `images/` - UI assets and documentation images
- `presets/` - Pre-configured layout presets
- `node_scripts/` - Backend scripts for license management
- `supabase/` - Database functions

## Usage

1. Select one or more faces in your SketchUp model
2. Click the PARAMETRIX toolbar icon
3. Configure your layout parameters
4. Apply the layout

## Requirements

- SketchUp 2017 or later
- Active internet connection for license validation

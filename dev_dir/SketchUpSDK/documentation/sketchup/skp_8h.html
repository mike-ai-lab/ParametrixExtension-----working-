Parameters
[in]file_pathThe model filepath to read from.
[out]guidThe guid string.
Returns
  • SU_ERROR_NONE on success
  • SU_ERROR_NULL_POINTER_INPUT if file_path is NULL
  • SU_ERROR_INVALID_INPUT if file_path does not exist
  • SU_ERROR_NULL_POINTER_OUTPUT if guid is NULL
  • SU_ERROR_INVALID_OUTPUT if guid does not point to a valid SUStringRef object.
  • SU_ERROR_MODEL_INVALID if the model is not valid
  • SU_ERROR_GENERIC if an unknown error occured reading the file
